# AnarchyItemGuard

Advanced protection plugin for Paper 1.16.5 Anarchy servers.

## Overview

AnarchyItemGuard prevents unauthorized use of configurable custom items inside WorldGuard regions.

The project is designed as part of a development portfolio focused on Minecraft Anarchy server administration and plugin development.

## Planned Features

- WorldGuard region integration
- Configurable protected items
- Owner and member permission checks
- Administrative bypass permission
- Custom messages
- Reload command
- Blocked interaction logging

## Requirements

- Paper or compatible server software for Minecraft 1.16.5
- WorldGuard 7
- WorldEdit 7
